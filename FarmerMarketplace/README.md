# 🌾 FarmerBuyer Marketplace

A smart digital marketplace platform that connects **farmers** directly with **buyers** without middlemen. The platform helps farmers sell agricultural products efficiently while allowing buyers to purchase fresh products directly from farmers.

---

# 🚀 Project Overview

FarmerBuyer Marketplace is a full-stack web application developed to simplify agricultural trading. The system provides separate dashboards for farmers, buyers, and administrators. Farmers can upload and manage products, buyers can browse and purchase products, and admins can monitor the overall platform.

The project also includes AI-powered functionalities such as prediction services and disease detection modules to enhance smart farming support.

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/seshaiyan/FarmerMarketplace.git
```

## 2️⃣ Move into the Project Folder

```bash
cd FarmerMarketplace
```

## 3️⃣ Create Virtual Environment

```bash
python -m venv .venv
```

## 4️⃣ Activate Virtual Environment

### Windows

```bash
.venv\Scripts\activate
```

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 6️⃣ Run the Application

```bash
python run.py
```
